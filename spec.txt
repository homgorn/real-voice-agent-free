|----------|---------|---------------|

| Мониторинг | Grafana + Prometheus | Визуализация метрик в реальном времени |
| Логирование | ELK Stack (Elasticsearch, Logstash, Kibana) | Полный цикл обработки логов |
| Аутентификация | Keycloak | Гибкое управление доступом |
| CI/CD | Jenkins + GitHub Actions | Автоматизация развёртывания |
| NLP | Rasa + Hugging Face Transformers | Распознавание интентов на русском |
| Синтез речи | Coqui TTS (open‑source) | Бесплатный аналог Google TTS |
| Распознавание речи | Whisper | Высокая точность, поддержка русского |

Схема работы (автоматизированный пайплайн):
1. Пользователь вводит описание бизнеса («кафе „Уют“, меню, бронирование»).
2. LangChain генерирует сценарий диалога.
3. Botpress визуализирует сценарий в конструкторе.
4. Система развёртывает навык через Kubernetes.
5. При запросе от Алисы:
* запрос → бэкенд VoiceBiz;
* Rasa обрабатывает интенты и сущности;
* ответ формируется и отправляется Алисе;
* диалог логируется в Elasticsearch.
6. Apache Airflow генерирует ежедневный отчёт в Telegram.
7. Grafana обновляет дашборд в реальном времени.

Оптимизации:
* AI‑генерация сценариев сокращает время настройки с 30 минут до 5 минут;
* Kubernetes снижает затраты на хостинг на 40 %;
* ELK Stack автоматизирует анализ логов (поиск ошибок, популярных команд).
